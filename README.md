# travel

## 本项目为 vue/cli 3.0 及以上版本，若要看 vue/cli 3.0 以下版本，可切换分支为 vue-cli3

## 项目为 去哪儿网的简单实现。一共三个页面
1. 首页
2. 城市列表页
3. 详情页

## 项目实现过程可看作者掘金 [去哪儿网复盘](https://juejin.cn/post/6983591357184999431/)

## Project setup
```
npm install
```
